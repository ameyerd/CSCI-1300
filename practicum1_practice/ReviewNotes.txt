Data Types:
1. int (for integers)
    - int myInt = 5;
2. char (for characters)
    - char myChar = 'c';
3. float (for floating-point numbers)
    - float myFloat = 4.4531;
4. double (for double precision floating-point numbers)
    - double myDouble = 4.4531;
